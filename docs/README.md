# Django Polls App

A simple polls application built with Django. Users can register, log in, view polls, vote, and see results. This project includes full Sphinx documentation and a Docker setup for easy deployment.

---

## Features

- User registration and authentication
- View list of polls
- Vote on a poll
- View poll results
- Fully documented using Sphinx
- Docker-ready for containerized deployment

---

## Prerequisites

- Python 3.10+  
- pip  
- virtualenv (optional but recommended)  
- Docker (optional, for containerized setup)
